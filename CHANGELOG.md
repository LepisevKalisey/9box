# Changelog

Все заметные изменения проекта будут фиксироваться в этом файле.

## [1.1.0] - 2025-11-28
- Добавлен вывод версии приложения на экране авторизации
- Введён CHANGELOG
- Удалён блок шагов «Список | Оценка | Итог»
- Удалён поиск по списку доступных сотрудников
- Добавлен флаг компании: запрет менеджерам добавлять сотрудников
- На сервере реализован запрет создания сотрудника при включённом флаге
- В админке добавлен тумблер настройки компании
- На экране общей матрицы администратор может удалять оценки сотрудников
- Пользовательский доступ к итоговой матрице отключён, матрица доступна только администратору

## [1.0.0] - 2025-11-28
- Базовая версия приложения: аутентификация, оценка сотрудников, матрица результатов, админ‑панель
